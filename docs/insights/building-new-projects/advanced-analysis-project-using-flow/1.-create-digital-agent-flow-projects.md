## Create a proxy flow project

1. Create DA project
2. Add transfer node
3. Select the target flow project created earlier
4. Connect the transfer node with the start node

> **INFO:** The proxy flow project serves as a middleman between the Insight project and the Digital Agent (flow) project. Unlike the "Selected project to connect" feature in the [insight flow connector parameter](), the proxy flow project can handle changes of the hash ID of the connected project, which happens after every (re)training of the target flow project -> proxy projects always point to the last trained and deployed version of Flow.
