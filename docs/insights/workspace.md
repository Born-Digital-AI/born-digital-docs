# Workspace

Master the fundamentals of the Digital Studio Insight Workspace

# Select your page or tap the navigation bar.

|  |  |
| --- | --- |
| [Design](../insights/workspace/design.md) | This module is for Insights configuration - specifying parameters and categories that are then grouped in the prompts and used for analysis. |
| [Upload](../insights/workspace/upload.md) | This module is for uploading data that is to be analysed. |
| [Process](../insights/workspace/process.md) | This module is for following the progress of the data processing and rerunning the analysis if any changes are made to the settings or new files are uploaded. |
| [Analyse](../insights/workspace/analyse.md) | This module displays the dashboards with analysed data and gives possibility to create/adjust the dashboards or data showed in player link. |
