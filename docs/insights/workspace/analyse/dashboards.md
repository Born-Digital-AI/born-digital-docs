# Dashboards

In this module, you can create and manage Kibana dashboards that display analyzed data.

![]()

On-screen elements:

- Conversation Dashboard: list displays dashboards that can be accessed 

- Name of the dashboard: once clicked on, the Kibana dashboard is displayed inside the window:

![]()
- + New Dashboard: button for creating a new Kibana dashboard

> **INFO:** Only the Organization administrator user can edit the dashboard.
