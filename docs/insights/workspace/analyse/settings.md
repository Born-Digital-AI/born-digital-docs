# Settings

This module is for adjusting showed fields in player (used for playing recordings or seeing html emails).

![]()

On-screen elements:

- Player metadata:

- Parameter: input field for specifying the path to the exact parameter in the existing JSON metadata file. 
Example: analysis.customer.accountNumber
- Label: input field serves to display the parameter in a human-readable format. 
Example: Account Number
- Save: floating button for saving the settings to the Player metadata
